## Installation

Para instalar los paquetes usados en el proyecto, puede utilizar los siguientes comandos:

```bash
git clone NicolasCaminos/ProductMannager.git
npm install
npm init -y
npm install express
npm install -g nodemon
npm install -g sass
npm install sweetalert2
npm start

```
